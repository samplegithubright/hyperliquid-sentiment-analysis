# Hyperliquid Trader Behavior vs Market Sentiment

## Objective
Analyze how Bitcoin Fear/Greed sentiment affects trader behavior and performance.

## Methodology
- Daily aggregation of trader metrics
- Sentiment alignment
- Behavioral segmentation
- Random Forest predictive modeling
- KMeans clustering

## Key Insights
1. Greed → higher leverage, lower average PnL
2. Fear → lower frequency but better win rate
3. High-size traders suffer larger drawdowns

## Strategy Ideas
- Reduce leverage during Greed
- Scale cautiously during Fear
